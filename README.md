# YJLaser 업무 프로그램 통합 관리

유진레이저 업무 자동화 및 관리를 위한 프로그램 모음입니다.
DXF 도면 처리부터 청구서 발행, 웹 기반 ERP/웹하드, PC 관리까지 업무 전반을 지원합니다.

---

## 프로그램 구성

| 폴더 | 프로그램 | 핵심 기능 | 기술 스택 |
|------|---------|----------|----------|
| [`레이저네스팅프로그램`](./레이저네스팅프로그램/) | 도면 자동 배치 | DXF 도면을 철판 위에 최적 배치하여 재료 낭비 최소화 | Python 3.8, PyQt5, ezdxf, Shapely |
| [`유진레이저목형 관리프로그램`](./유진레이저목형%20관리프로그램/) | 청구서 발행/발송 | DXF 파일명에서 정보 추출 → 청구서 생성 → FAX/MMS/Email 자동 발송 | Python, PyQt5, Popbill API, openpyxl |
| [`yjlaser_website`](./yjlaser_website/) | 통합 웹 플랫폼 | 홈페이지, 관리자 대시보드, 거래처 대시보드, ERP, 웹하드 | Next.js 15, NestJS, Supabase, Cloudflare R2 |
| [`외부웹하드동기화프로그램`](./외부웹하드동기화프로그램/) | LGU+ 웹하드 자동 동기화 | LGU+ 외부 웹하드 파일을 자동 감지/다운로드하여 자체 웹하드로 동기화 | Node.js, TypeScript, Playwright, SQLite |
| [`computeroff`](./computeroff/) | PC 부팅/종료 모니터링 | 사무실 PC들의 부팅/종료 이벤트 수집 및 웹 대시보드 표시 | Python, FastAPI, SQLite |
| [`잡다한것`](./잡다한것/) | 기타 유틸리티 | 기타 도구 및 문서 | - |

## 워크플로우 (업무 프로세스)

```
거래처 문의 → 도면수정 → 컨펌 → 도면가공 → 파일분류 → 네스팅 → 레이저가공 → 오시칼작업 → 납품
    (1)        (2)      (3)      (4)     (4-1)     (5)     (5-1)      (6)       (7)
```

### 1. 일 접수 (두 가지 경로)

**1-1. 회사사이트 문의**
- 거래처가 회사 웹사이트에서 문의 등록 (도면 첨부)
- (계획) 자동으로 자체 웹하드에 업체별 폴더 생성 및 파일 등록
- 담당 프로그램: [`yjlaser_website`](./yjlaser_website/)

**1-2. 외부웹하드 업로드 + 유선 전달**
- 거래처가 LGU+ 웹하드 `올리기전용/{업체명}/` 폴더에 도면 업로드
- 유선(전화)으로 작업 내용 전달
- `외부웹하드동기화프로그램`이 자체 웹하드로 자동 동기화
- 담당 프로그램: [`외부웹하드동기화프로그램`](./외부웹하드동기화프로그램/)

**외부웹하드 폴더 구조:**
```
LGU+ 웹하드/
├── 올리기전용/          ← 거래처 → 유진레이저 (도면 전달)
│   ├── 업체A/
│   ├── 업체B/
│   └── ...
└── 내리기전용/          ← 유진레이저 → 거래처 (수정도면/컨펌)
    ├── 업체A/
    ├── 업체B/
    └── ...
```

**자체웹하드 폴더 구조** (외부웹하드와 1:1 미러링):
```
자체웹하드/
├── 올리기전용/          ← 외부웹하드에서 자동 동기화
│   ├── 업체A/
│   └── 업체B/
└── 내리기전용/          ← 수정 도면 업로드 → 외부로 역동기화
    ├── 업체A/
    └── 업체B/

※ 파일별 작업 상태(접수/수정중/컨펌/완료)는 웹사이트 DB에서 관리
```

### 2. 도면 수정

- 문의 내용 및 제공 도면을 바탕으로 샘플 제작 및 제품 생산을 위한 도면 수정
- 도무송 규격에 맞게 CAD 프로그램으로 가공
- 수동 작업 (프로그램 연동 없음)

### 3. 수정 도면 전달 및 컨펌

- CAD로 수정한 도면을 외부웹하드 `내리기전용/{업체명}/` 폴더에 업로드
- 거래처가 도면 확인 후 컨펌
- 컨펌 후 거래처가 목형 제작을 위한 최종 도면을 `올리기전용/{업체명}/` 폴더에 업로드
- 수동 프로세스 (유선/메신저 소통)

### 4. 도면 가공 및 작업폴더 이동

- 도면작업자가 자체 웹하드에서 최종 컨펌된 도면 다운로드
- 목형 제작을 위한 도면 가공 (CAD)
- 가공 완료된 DXF 파일을 네트워크 작업폴더로 이동

**작업폴더 경로:** `\\192.168.0.6\home\데이터\유진MAIN\dxf\`

**파일명 규칙:** `{MMDD}-{순번} {업체명} {제품설명} {규격}.DXF`
- 예: `0219-1 원컴퍼니 [어뮤즈][ONE-26-54] 퀘이크업미니가차_목형의뢰 [우주팩 107] 2절.DXF`
- 예: `0219-4 대성목형(29927).DXF`
- 예: `0219-5 신영 박누가 달밤의꿀 단상자 545-444.DXF`
- `00-`으로 시작하는 파일은 테스트/샘플 파일

**폴더 구조:** 날짜/업체 구분 없이 한 폴더에 모든 작업 파일 보관

### 4-1. 관리프로그램 자동분류

- DXF 작업폴더의 파일명에서 업체명, 가격 자동 추출
- 업체별/월별 폴더로 자동 복사 정리
- 가격 삽입 기능 (파일명에 가격 추가/수정)
- 업체별 청구서 자동 생성 (Excel)
- 청구서 자동 발송 (FAX/MMS/Email via Popbill API)
- 담당 프로그램: [`유진레이저목형 관리프로그램`](./유진레이저목형%20관리프로그램/)

```
DXF 작업폴더 (한 폴더에 모든 파일)
    ↓ 파일명에서 업체명/가격 추출
네트워크 정리폴더/
├── 원컴퍼니/
│   └── 2026년02월/
│       ├── 0219-1 원컴퍼니...DXF
│       └── 0219-2 원컴퍼니...DXF
├── 대성목형/
│   └── 2026년02월/
└── 신영/
    └── 2026년02월/
    ↓
업체별 청구서 자동 생성 및 발송
```

### 5. 레이저 네스팅

- 네스팅 프로그램이 동일 DXF 작업폴더를 감지
- 새로 추가된 DXF 파일을 자동으로 네스팅 (합판 배치 최적화)
- 재료 사용률 극대화를 위한 자동 배치 알고리즘
- 담당 프로그램: [`레이저네스팅프로그램`](./레이저네스팅프로그램/)

### 5-1. 레이저 가공

- 작업자가 네스팅 결과를 확인 후 '배치완료' 클릭
- 네스팅된 파일 목록이 네트워크 공유 Excel에 자동 저장
- 작업자가 레이저가공기로 실제 절단 작업 시작
- 가공 완료 후 목형 반제품 생산

### 6. 오시칼 작업

- 레이저 가공된 목형에 오시(접선)/칼(절단선) 수작업 진행
- 현재: 프로그램 연동 없는 수동 작업
- 향후: 자동화 계획 있음

### 7. 납품

- 대부분 직접 배달로 납품
- 일부 화물 발송
- 웹사이트 납품관리 페이지에서 관리

| 5. 네스팅 | 구현됨 | 자동 |
| 5-1. 레이저가공 Excel | 구현됨 | 반자동 |
| 5-1. 레이저가공 실행 | 수동 | 수동 |
| 6. 오시칼 작업 | 수동 | 수동 (자동화 계획) |
| 7. 납품 | 웹사이트 관리 | 반자동 |
